# RestaurantsApp

A Spring Boot REST API application to manage Restaurants and Reviews with user roles and authentication.

## Features
- Add, update, and list restaurants
- Add and approve reviews
- Role-based authentication (`ADMIN` and `USER`)
- Validation for all fields
- In-memory user authentication for testing
- Spring Boot, JPA, Hibernate, MySQL integration

## Technologies Used
- Java 17
- Spring Boot 3
- Spring Security
- JPA/Hibernate
- PstGresSql
- Maven

## Getting Started

### Prerequisites
- Java 17
- Maven
- MySQL database
- IDE (IntelliJ)
